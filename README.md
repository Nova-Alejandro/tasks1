# Simple Task Manager
A simple console program to organize your daily tasks. This project allows you to manage a to-do list quickly and easily.

# Features
The code features an interactive menu with the following options:
Add Task: Allows you to write a new task and save it to the current list. Ideal for capturing tasks instantly.
Delete Task: Displays the numbered list so you can choose which one to delete. Useful for clearing completed tasks or errors.
View Tasks: Displays all your pending tasks on the screen so you don't forget any.
Exit: Closes the program safely, ending the script's execution.
